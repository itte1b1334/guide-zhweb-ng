# guide-zhweb-ng

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或具体网站，仅作为通用 HTML 文档的存储与分发用途。

## 项目简介

本项目旨在提供一个结构清晰、易于访问的 HTML 页面集合。仓库中的每个页面均为独立文件，可直接在浏览器中打开查看，无需依赖外部服务器环境。

所有页面均遵循标准 HTML 规范，适合用于学习、参考、测试或临时展示场景。

## 目录说明

```
.
├── index.html       # 入口页面（可选）
├── pages/           # 存放主要 HTML 页面
│   ├── page1.html
│   └── page2.html
├── assets/          # 公共资源（CSS、JS、图片等）
│   ├── styles/
│   └── scripts/
└── README.md        # 本文件
```

- `pages/`：各独立 HTML 页面存放于此目录。
- `assets/`：页面共用的样式、脚本、图片等资源文件。
- 根目录下的 `index.html` 可作为导航或入口页面（若存在）。

## 页面归档说明

- 每个页面均为自包含的 HTML 文档，可直接通过浏览器打开。
- 页面间无强依赖关系，可独立使用。
- 部分页面可能引用 `assets/` 中的公共资源，请确保目录结构完整。
- 页面文件名及内容可能随仓库更新而变化，请以最新版本为准。
